# gitlab-runner

GitLab Runner is a tool that allows you to run your CI/CD jobs and send the results back to GitLab.

## Requirements
In order to use this port you will need the following APARS with thes PTFs:

- z/OS V2.5
  - OA62757
  - PH45182

- z/OS V3.1:
  - OA62757
  - PH57159
